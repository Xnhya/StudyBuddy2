Consultar DNI

Información Técnica
Endpoint de la API
GET /api/external/dni/{dni}
Ejemplo de Uso
// Request
GET /api/external/dni/71234567

// Response (éxito)
{
  "success": true,
  "data": {
    "dni": "71234567",
    "nombres": "JUAN CARLOS",
    "apellidoPaterno": "PEREZ",
    "apellidoMaterno": "GARCIA"
  }
}
Características
  • Validación de formato DNI (8 dígitos)
  • Cache de consultas para mejor rendimiento
  • Integración con API externa de RENIEC
  • Límite: 10 consultas por minuto
Códigos de Respuesta
200 Consulta exitosa
400 DNI inválido o formato incorrecto
404 DNI no encontrado
500 Error del servidor
Acerca de la Consulta DNI

Este servicio permite consultar información de DNI mediante una API externa.

Características:

  • Validación de formato DNI (8 dígitos)
  • Cache de consultas para mejor rendimiento
  • Integración con API externa de RENIEC
  • Límite de 10 consultas por minuto
